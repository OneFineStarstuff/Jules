# Interactive AGI/ASI Framework Explorer: Functional Specification

## 1. Objective
A React-based interactive visualizer for mapping AGI capability levels to required governance interventions.

## 2. Core Components
*   **Capability Slider:** Users toggle between "Weak AI" and "ASI Readiness."
*   **Governance Heatmap:** Real-time visualization of risk (Red/Amber/Green) across the 18-component model.
*   **Activation Simulator:** Simulates "Hard Kill" (IRMI) vs. "Soft Gating" (GDL) scenarios.

## 3. Technical Stack
*   **Frontend:** React + D3.js for graph visualization.
*   **Governance Engine:** Open Policy Agent (OPA) for logic simulation.
